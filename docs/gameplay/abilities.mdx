## Ability Scaling

All abilities scale from **derived stats**, not primary stats directly:

| Class | Abilities Scale From |
|-------|---------------------|
| <ClassMention class="paladin">Paladin</ClassMention> | <DerivedStatMention stat="physical_attack" link={false}>Physical Attack</DerivedStatMention> (+ <DerivedStatMention stat="magic_attack" link={false}>Magic Attack</DerivedStatMention> for hybrid abilities) |
| <ClassMention class="cleric">Cleric</ClassMention> | <DerivedStatMention stat="magic_attack" link={false}>Magic Attack</DerivedStatMention> |
| <ClassMention class="warrior">Warrior</ClassMention> | <DerivedStatMention stat="physical_attack" link={false}>Physical Attack</DerivedStatMention> |
| <ClassMention class="ranger">Ranger</ClassMention> | <DerivedStatMention stat="physical_attack" link={false}>Physical Attack</DerivedStatMention> |
| <ClassMention class="mage">Mage</ClassMention> | <DerivedStatMention stat="magic_attack" link={false}>Magic Attack</DerivedStatMention> |

<Callout type="info">
Equipment directly affects ability damage. A sword with +50 <DerivedStatMention stat="physical_attack" link={false}>Physical Attack</DerivedStatMention> adds +50 × skill multiplier to Warrior abilities!
</Callout>

<Callout type="tip">
**Reading Ability Stats:** Values shown as "80→250" indicate Level 1 → Level 10 progression. See individual class pages for complete level-by-level scaling tables.
</Callout>
