## Area Boss Classification

| Attribute | Value |
|-----------|-------|
| **HP Multiplier** | 15x base creature |
| **Damage** | 5x base creature |
| **Party Size** | 2-4 players |
| **Respawn Time** | 2 hours |
| **Reward Tier** | High-value drops (20% Prestige) |

---

## Mechanics

Area Bosses have **2 abilities + 1 passive** that define their combat style:

- **Primary Ability** - Main damage dealing attack
- **Secondary Ability** - Additional offensive capability
- **Passive** - Debuff aura or on-hit effect that pressures the party
