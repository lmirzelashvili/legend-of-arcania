Shields are defensive equipment used exclusively by <ClassMention class="paladin">Paladins</ClassMention>. Each shield uses the **hybrid random stat system** with base defense, identity stat, random bonus stats, and sockets for gems.

---

## Shield Stat Model

Each shield provides:

| Component | Description |
|-----------|-------------|
| **Defense** | Base defense value (scales by tier, ~±10% variance) |
| **Identity Stat** | Physical Defense (guaranteed) |
| **Prestige Stat** | Prestige Block % (only on Prestige quality drops) |
| **Random Bonus Stats** | 1-3 stats from defensive pool |
| **Sockets** | Empty slots for gems (1/1/2/2/3 by tier) |

<Callout type="info">
<ClassMention class="paladin" link={false}>**Paladin Only:**</ClassMention> Shields can only be equipped by Paladins, providing them with unique tanking capabilities.
</Callout>

## Prestige Shields

Shields have a 10% chance to drop as **Prestige** quality, gaining bonus Block chance.

| Tier | Prestige Block % |
|------|------------------|
| Tier 1 | +1% |
| Tier 2 | +2% |
| Tier 3 | +3% |
| Tier 4 | +4% |
| Tier 5 | +6% |

<Callout type="info">
Prestige Block % increases your chance to block incoming attacks, reducing damage taken. Essential for tank Paladins.
</Callout>
