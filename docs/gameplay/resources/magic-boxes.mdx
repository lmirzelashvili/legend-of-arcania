title: Magic Boxes
description: Tiered loot boxes containing equipment, gold, materials, and other valuable resources.

Magic Boxes are mysterious containers that hold randomized rewards. Higher tier boxes contain better equipment and more valuable resources.

## Acquisition Sources

| Source | Available Boxes |
|--------|-----------------|
| **Dominion Bosses** | Ember, Inferno, Eclipse |
| **Area Bosses** | Cinder, Ember, Inferno |
| **Apex Creatures** | Ember, Inferno |
| **Elite Creatures** | Ash, Cinder, Ember |
| **Hostile Creatures** | Ash, Cinder |
| **Daily Rewards** | Ash, Cinder |
| **Quest Rewards** | All tiers (quest-dependent) |
| **Events** | Special event boxes |

<Callout type="tip">
Higher tier boxes have better drop rates for equipment but remain rare. Focus on Dominion content for consistent access to Inferno and Eclipse boxes.
</Callout>
