## Market Opportunity

### Target Demographics

| Segment | Age Range | Size | Characteristics |
|---------|-----------|------|-----------------|
| Primary | 25-40 | Large | Played classic MMOs, disposable income, nostalgia-driven |
| Secondary | 18-24 | Medium | New to genre, seeking accessible entry point |
| Tertiary | Mobile users | Growing | Future expansion opportunity |

### Competitive Landscape

Arcania occupies a unique position:

- **vs. Modern MMOs (WoW, FFXIV):** Simpler, browser-based, no subscription
- **vs. Classic private servers:** Legal, maintained, quality-of-life features
- **vs. Mobile MMOs:** Not pay-to-win, actual gameplay (not idle)
- **vs. Browser games:** True MMO depth, not casual farming game

## Monetization Philosophy

<Callout type="success" title="Fair Play Commitment">
Arcania is committed to ethical monetization. Players can never buy direct power advantages.
</Callout>

### Revenue Streams

**Premium Currency: Void Crystals**

| Category | Examples | Price Range |
|----------|----------|-------------|
| Cosmetics | Skins, wings, effects | $2-15 |
| Convenience | Inventory slots, teleports | $1-5 |
| Time savers | XP boosts (not power) | $1-3 |
| Character services | Name change, transfers | $5-10 |

### What We DON'T Sell

- Direct stat advantages
- Best-in-slot equipment
- Exclusive abilities
- Gameplay-affecting power

### Free-to-Play Accessibility

Players can earn premium currency through:
- Achievement milestones
- Seasonal events
- Trading with other players
- Daily login bonuses

## Technical Architecture

### Stack Overview

| Layer | Technology | Rationale |
|-------|------------|-----------|
| Frontend | React + TypeScript | Industry standard, large talent pool |
| Backend | Node.js + Express | Real-time capable, scalable |
| Database | PostgreSQL | Reliable, feature-rich |
| Real-time | Socket.IO | WebSocket with fallbacks |
| ORM | Prisma | Type-safe, developer-friendly |

### Scalability

- Horizontal scaling via load balancers
- Database sharding strategy planned
- CDN for static assets
- Regional servers for latency optimization
