Arcania uses two currencies: **Gold** for everyday gameplay and **Arcanite** as a scarce premium currency convertible to **ARC Token**.

---

## Currencies

| Currency | Purpose | Supply |
|----------|---------|--------|
| **Gold** 💰 | Everyday gameplay | Unlimited (monsters, quests, trading) |
| **Arcanite** <img src="/game-icons/arcanite.png" alt="Arcanite" className="inline w-4 h-4 align-middle" /> | Premium features | Scarce (purchase, rare drops, sinks) |

### Gold

Everyday currency with unlimited supply:
- NPC vendors, repairs, enhancements
- Marketplace trading
- Vault expansion

### Arcanite

Scarce currency with sink mechanics:
- Cosmetics, boosters, premium features
- Name changes, premium crafting
- **Marketplace sales** (sell items for Arcanite)
- **ARC Token conversion** (cryptocurrency)
