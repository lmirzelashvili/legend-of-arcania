## Rating Calculation

### Kill Value

Not all kills are equal:

| Kill Type | Points | Reason |
|-----------|--------|--------|
| Equal level | 10 | Base value |
| +1-5 levels | 12-15 | Harder opponent |
| +6-10 levels | 16-20 | Significant challenge |
| -1-5 levels | 8-5 | Easier opponent |
| -6-10 levels | 4-2 | Much easier |
| -11+ levels | 1 | Griefing deterrent |

### Death Value

| Death Type | Point Loss |
|------------|------------|
| To equal level | -8 |
| To higher level | -5 to -3 |
| To lower level | -10 to -15 |
| To same player (repeated) | -2 (farming deterrent) |

### Rating Formula

```
Rating Change = (Kill Points Earned) - (Death Points Lost) × Streak Modifier

Streak Modifier:
• 3+ kill streak: ×1.1
• 5+ kill streak: ×1.25
• 10+ kill streak: ×1.5
```

## Anti-Abuse Measures

### Kill Trading Prevention

| Behavior | Consequence |
|----------|-------------|
| Killing same player repeatedly | Diminishing returns (1 point after 3rd) |
| Suspected boosting | Rating penalty, investigation |

### Alt Account Detection

- Same IP kills flagged
- Pattern detection for boosting
- Manual review for top rankings

<Callout type="warning">
**Fair Play:** Attempting to manipulate rankings through kill trading, boosting, or alt accounts results in rating penalties and potential account action.
</Callout>
