Add players to your friends list to stay connected across Nexus and in-game.

---

## Features

| Feature | Description |
|---------|-------------|
| **Online Status** | See when friends are online |
| **Quick Invite** | One-click party invites |
| **Whisper** | Direct messaging |
| **Location** | See which zone friends are in |

---

## Managing Friends

### Adding Friends

- Right-click player name → Add Friend
- `/friend add [PlayerName]`

### Removing Friends

- Open Friends list → Right-click → Remove
- `/friend remove [PlayerName]`

---
