**ARC Token** is the native cryptocurrency of Legend of Arcania, designed for long-term sustainability with a fixed maximum supply and controlled emission schedule. Unlike inflationary game tokens that collapse under selling pressure, ARC implements multiple deflationary mechanisms to maintain value.

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Fixed Supply** | 100,000,000 maximum tokens, never increased |
| **Earned, Not Bought** | Primary acquisition through gameplay |
| **Deflationary** | Multiple burn mechanisms reduce circulating supply |
| **Utility-Driven** | Real in-game and governance uses |

---

## Token Overview

ARC Token serves multiple purposes within the Arcania ecosystem:

### For Players
- Convert earned Arcanite to real cryptocurrency
- Trade on decentralized exchanges
- Access exclusive token-holder features
- Participate in governance decisions

### For the Ecosystem
- Marketplace transaction currency
- Staking for enhanced rewards
- DAO governance voting power
- Premium feature access

---

## Acquisition Methods

| Method | Description | Availability |
|--------|-------------|--------------|
| **Arcanite Conversion** | Exchange in-game Arcanite for ARC | Level 35+ |
| **DEX Trading** | Purchase on decentralized exchanges | Public |
| **Staking Rewards** | Earn additional ARC by staking | Token holders |
| **Special Events** | Limited-time earning opportunities | All players |

<Callout type="tip">
The most rewarding path to ARC tokens is through dedicated gameplay. Veteran players with max account rates receive significantly better conversion ratios than market purchases.
</Callout>
