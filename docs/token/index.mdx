Legend of Arcania integrates blockchain technology to create a **sustainable Play-to-Earn economy** where dedicated players can earn real value while enjoying the game. Our dual-currency system keeps core gameplay accessible while offering crypto rewards for committed players.

---

## Design Philosophy

Arcania's token economy is built on lessons learned from the P2E industry:

- **Gameplay First** - The game must be fun without any earning potential
- **Sustainable Emissions** - Fixed halving schedule with 100M hard cap
- **Anti-Extraction Design** - Systems reward dedication, not exploitation
- **Deflationary Pressure** - Multiple sink mechanisms maintain token value

---

## Dual-Currency System

| Currency | Type | Purpose |
|----------|------|---------|
| **Gold** | Off-chain | Everyday gameplay transactions |
| **Arcanite** | Bridged | Premium currency, convertible to ARC |
| **ARC Token** | On-chain | Blockchain token with real-world value |

Gold remains entirely off-chain, ensuring core gameplay is never affected by market volatility. Arcanite serves as the controlled bridge between in-game economy and the ARC Token.

## Quick Facts

| Metric | Value |
|--------|-------|
| **Token Name** | ARC Token |
| **Max Supply** | 100,000,000 ARC |
| **Play-to-Earn Allocation** | 75% (75M tokens) |
| **Base Conversion** | 100 Arcanite = 1 ARC |
| **Minimum Requirements** | Level 35, 100h over 30+ days |
