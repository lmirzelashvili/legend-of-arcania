## The Path to the Throne

```
Valoryn → Dreadmar Frontier → Mire of Whispers → Stonegrave Expanse → Eclipse Barrens → Hall of Fallen → Throne of Ruin
```

Each dominion must be conquered in sequence. The generals are bound to their territories-they cannot be bypassed, only defeated.

## Strategic Importance

Each dominion serves a purpose beyond simply blocking the path:

- **Dreadmar Frontier** - Tests raw combat ability and survival skills
- **Mire of Whispers** - Tests mental fortitude and party cohesion
- **Stonegrave Expanse** - Tests tactical coordination and resource management
- **Eclipse Barrens** - Tests faith and willpower against despair
- **Hall of Fallen** - Tests everything, including the meaning of victory
